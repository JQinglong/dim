# dim
Data Installation Manager: Manage the open data etc. in your project like a package manager.


# Usage

```
dim init
```

```
dim install [open data url] -n [name]
```

# Commands

## Init

```
dim init
```

## Install

```
dim install [open data url] -n [name]
```

## Uninstall

```
dim uninstall [name or url]
```

## List

```
dim list
```

## Update

```
dim update [name or url]
```

## Upgrade

```
dim upgrade
```

## Help

```
dim help
```
